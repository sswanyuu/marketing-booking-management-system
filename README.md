### Prerequisites
- Node.js (v16 or higher)

### Installation

1. Clone the repository
```bash

cd booking-management-system
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run docs:dev
```

4. Build for production
```bash
npm run docs:build
```
